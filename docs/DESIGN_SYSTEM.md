# AidiCore Design System V1

## Brand

AidiCore is positioned as a modern technology platform for community impact, not a traditional charity website.

## Logo Direction

The included SVG concept combines:
- A letterform
- Heart-like emotional softness
- Ripple effect for impact propagation

The original uploaded logo files are kept in `src/assets/` as reference assets.

## Colors

- Deep background: `#020617`
- Navy surface: `#0F172A`
- Trust blue: `#2563EB`
- Emerald impact: `#10B981`
- Gold success: `#F59E0B`

## Typography

- English: Inter
- Arabic: Cairo

## UI Style

- Dark premium interface
- Rounded cards
- Glass navigation
- Clear CTAs
- Mobile-first responsive layout
- No social-media-like competitive leaderboard
